# Recycling Center System (Java)

## 📌 About
This is a learning project developed to improve my programming skills.  
This project simulates a recycling center system using Java, focusing on data structures and object-oriented programming.

## 🚀 Features
- Vehicle queue management system
- Waste tracking and processing
- Customer management
- Use of data structures (Queue, LinkedList)
- System simulation logic

## 🛠️ Technologies
- Java
- Data Structures
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Open the project in an IDE (IntelliJ IDEA / Eclipse)
2. Run the `Main` class

## 🎯 Purpose
This project was created to practice data structures and simulate a real-world system scenario.
